# Feature Spec: RGBキューブ表示

## 1. 背景

- Three.js ベースの描画基盤を安定化するため、RGB空間を3次元で表示する
- 本機能は技術MVPであり、写真分析機能を実装する前提基盤として扱う

## 2. スコープ

### In Scope

- RGB色空間を3Dキューブとして表示する
- 回転操作を提供する
- キューブ解像度は256で固定する
- Slice表示と同一の探索コンテキストで扱える情報配置を定義する
- 画像由来のマッピングと選択色マッピングの表示を個別に切り替える

### Out of Scope

- HSL / L*C*h など他色空間の3D表示
- 写真分析（Lab散布図、ヒストグラム、面積比）
- レンダリング品質の詳細設定（影/反射など）
- ズーム/パン操作
- 解像度の切り替え（スライダー選択）

## 5. 機能要求

### FR-1: RGBキューブを3D表示できる

#### 受け入れ条件

- 3D表示領域にRGBキューブが初期表示される
- RGBの各軸方向が視覚的に区別できる

### FR-2: ユーザーはキューブを操作できる

#### 受け入れ条件

- ドラッグで回転できる

### FR-3: キューブ解像度は256で固定する

#### 受け入れ条件

- 初期表示の解像度は256である
- 解像度を変更する操作は提供しない

### FR-4: キューブとSliceの往復が短い配置である

#### 受け入れ条件

- ワークベンチ上でRGBキューブとSlice表示が近接して配置される
- キューブ操作中に、同一画面内で視線移動を最小化してSlice状態を確認できる

### FR-5: キューブ上の補助マッピング表示を切り替えできる

#### 受け入れ条件

- 画像由来の白マッピングを表示/非表示できる
- 選択色のオレンジマッピングを表示/非表示できる
- 表示切替 UI は折りたたみ内に配置される
