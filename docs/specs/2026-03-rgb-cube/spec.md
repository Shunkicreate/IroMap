# Feature Spec: RGBキューブ表示

## 1. 背景
- 色空間を直感的に理解するためには、RGB空間を3次元として観察できる必要がある
- 色の分布や変化を立体的に把握できる表示がMVPの核となる

## 2. スコープ
### In Scope
- RGB色空間を3Dキューブとして表示する
- 回転/ズーム/パンの基本操作を提供する
- キューブ解像度（サンプル密度）を切り替えられる

### Out of Scope
- HSL / L*C*h など他色空間の3D表示
- 画像の色分布プロット
- レンダリング品質の詳細設定（影/反射など）

## 5. 機能要求
### FR-1: RGBキューブを3D表示できる
#### 受け入れ条件
- 3D表示領域にRGBキューブが初期表示される
- RGBの各軸方向が視覚的に区別できる

### FR-2: ユーザーはキューブを操作できる
#### 受け入れ条件
- ドラッグで回転できる
- スクロールでズームできる
- 右ドラッグまたは専用操作でパンできる

### FR-3: キューブ解像度を切り替えられる
#### 受け入れ条件
- 解像度プリセットから選択できる（例: 8/16/32）
- 切り替え後に表示が更新される
