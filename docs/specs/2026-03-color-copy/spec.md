# Feature Spec: カラーコピー

## 1. 背景

- 探索した色を外部で利用するため、値のコピーが必要
- 形式を切り替えて使えることが実務上の利便性に直結する

## 2. スコープ

### In Scope

- クリックした色を選択色として固定する
- 形式を切り替えてコピーできる（HEX / rgb() / hsl()）
- 探索導線を阻害しない情報配置を定義する

### Out of Scope

- 複数色の同時コピー
- パレット管理
- クリップボード履歴管理

## 5. 機能要求

### FR-1: クリックした色を選択色として固定できる

#### 受け入れ条件

- 3DまたはSlice上のクリックで選択色が更新される
- 選択色はインスペクタ領域とは別に保持される

### FR-2: 形式を切り替えてコピーできる

#### 受け入れ条件

- HEX / rgb() / hsl() の形式を選択できる
- 選択中の形式の文字列がコピーできる

### FR-3: コピーUIは探索導線から分離できる

#### 受け入れ条件

- RGBキューブ/2D断面の探索UIより後段の位置にコピーUIを配置できる
- 色確定後にコピー操作へ遷移しやすい構造である
