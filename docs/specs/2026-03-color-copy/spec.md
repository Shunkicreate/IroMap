# Feature Spec: カラーコピー

## 1. 背景

- 探索した色を外部で利用するため、値のコピーが必要
- 形式を切り替えて使えることが実務上の利便性に直結する

## 2. スコープ

### In Scope

- クリックした色を選択色として固定する
- HEX / rgb() / hsl() を個別にコピーできる
- クリップボードから色文字列を貼り付け、選択色へ反映できる
- 探索導線を阻害しないよう、インスペクタ内へ統合配置する

### Out of Scope

- 複数色の同時コピー
- パレット管理
- クリップボード履歴管理

## 5. 機能要求

### FR-1: クリックした色を選択色として固定できる

#### 受け入れ条件

- 3DまたはSlice上のクリックで選択色が更新される
- 選択色はインスペクタ領域とは別に保持される

### FR-2: 各形式を個別にコピーできる

#### 受け入れ条件

- HEX / rgb() / hsl() の各表示値に対応したコピー操作を持てる
- クリックした形式の文字列をコピーできる

### FR-3: コピーUIは探索導線を阻害しない

#### 受け入れ条件

- コピーUIはインスペクタと同一パネル内にまとまり、探索UIを圧迫しない
- 色確定後にコピー操作へ遷移しやすい構造である

### FR-4: クリップボード文字列から選択色を復元できる

#### 受け入れ条件

- 貼り付け操作で HEX / rgb() / hsl() 文字列を読み取り、選択色へ反映できる
- 未対応形式の文字列入力時にユーザーへ失敗メッセージを表示する
