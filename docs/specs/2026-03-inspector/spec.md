# Feature Spec: カラーインスペクタ

## 1. 背景

- 探索中の色を数値として把握できることで学習効果が高まる
- 3D/2D表示から即座に色情報を参照できる必要がある
- 一時プレビュー（ホバー）と選択色（クリック）を区別することで操作意図を明確にする

## 2. スコープ

### In Scope

- ホバー中の色をインスペクタ領域に表示する（プレビュー）
- クリックした色を選択色として保持し表示する（選択）
- RGB/HEX/HSLの3形式を同時に表示する
- プレビュー要素と選択要素を比較しやすい配置で表示する

### Out of Scope

- 色履歴の保存
- 変換精度の詳細な比較表示
- 他色空間（CIELAB等）の表示

## 5. 機能要求

### FR-1: ホバー中の色を表示できる（プレビュー）

#### 受け入れ条件

- 3D表示またはSlice表示のホバーでプレビュー色が更新される
- ホバーが外れた場合はプレビューが未選択状態に戻る

### FR-2: クリックした色を選択色として保持できる

#### 受け入れ条件

- 3D表示またはSlice表示のクリックで選択色が更新される
- ホバーが外れても選択色は保持される

### FR-3: RGB/HEX/HSLを同時に表示できる

#### 受け入れ条件

- 3形式の値が同時に表示される
- 表示値は同一色に対して整合している

### FR-4: プレビューと選択を横並びで表示できる

#### 受け入れ条件

- デスクトップ幅では、プレビュー要素と選択要素が横並びで表示される
- 2要素の比較時に視線移動が最小化される
- モバイル幅では可読性を優先して縦並びへフォールバックできる
