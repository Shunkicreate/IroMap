# Feature Spec: カラーインスペクタ

## 1. 背景
- 探索中の色を数値として把握できることで学習効果が高まる
- 3D/2D表示から即座に色情報を参照できる必要がある

## 2. スコープ
### In Scope
- ホバー中の色をインスペクタ領域に表示する
- RGB/HEX/HSLの3形式を同時に表示する

### Out of Scope
- 色履歴の保存
- 変換精度の詳細な比較表示
- 他色空間（CIELAB等）の表示

## 5. 機能要求
### FR-1: ホバー中の色を表示できる
#### 受け入れ条件
- 3D表示またはSlice表示のホバーで色が更新される
- ホバーが外れた場合は未選択状態が表示される

### FR-2: RGB/HEX/HSLを同時に表示できる
#### 受け入れ条件
- 3形式の値が同時に表示される
- 表示値は同一色に対して整合している
