# Test Cases: ユーザー登録

## 対応要求
- FR-1
- FR-2

## 正常系
- TC-001: 有効な email/password で登録申請すると 202 を返す（FR-1）
- TC-002: 登録成功時に pending user が作成される（FR-1）
- TC-003: 30 分有効な確認リンクを含むメールが送信される（FR-2）

## 異常系
- TC-101: メール形式不正で 400 を返す（FR-1）
- TC-102: パスワード 12 文字未満で 400 を返す（FR-1）
- TC-103: 重複メールで汎用メッセージを返し新規作成しない（FR-1）
- TC-104: 期限切れ token でメール確認に失敗する（FR-2）
- TC-105: 60 秒以内の再送要求を拒否する（FR-2）
