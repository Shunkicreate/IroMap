# Test Cases: Photo RGB Cube Mapping

## 対応要求

- FR-1
- FR-2
- FR-3
- FR-4
- FR-5

## 正常系

- TC-001: 一般的な写真をアップロードすると、RGB キューブ上に画像由来の点群が表示される（FR-1）
- TC-002: 表示モードを `grid only` / `image only` / `grid + image` に切り替えられる（FR-2）
- TC-003: 画像を差し替えると、RGB キューブ上の点群も同じ画像に更新される（FR-3）
- TC-004: 高解像度画像でも点群数が制御され、操作継続可能な描画性能を保つ（FR-4）
- TC-005: 単色または偏色画像で、頻出色が視認しやすい形で表示される（FR-5）

## 異常系

- TC-101: 画像未選択の状態で `image only` を選択しても表示が破綻しない（FR-2）
- TC-102: 画像読み込み失敗時に RGB キューブへ不正な点群が残らない（FR-1, FR-3）
- TC-103: 透明ピクセルのみの画像入力時に空状態を維持できる（FR-1, FR-4）
