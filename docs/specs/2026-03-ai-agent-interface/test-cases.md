# Test Cases: AI Agent対応画像解析インターフェース

## 対応要求

- FR-1
- FR-2
- FR-3
- FR-4
- FR-5
- FR-6
- FR-7
- FR-8

## 正常系

- TC-001: ブラウザから画像アップロードして解析結果HTMLを表示できる（FR-1, FR-2）
- TC-002: APIへ raw image body をPOSTしてJSON解析結果を取得できる（FR-1, FR-3）
- TC-003: ブラウザアップロードと raw body API で同一画像バイト列の結果が一致する（FR-1, FR-3）
- TC-004: HTML結果に主要色、平均明度、平均彩度、特徴文が含まれる（FR-2, FR-5）
- TC-005: JSON結果に `input` `summary` `analysis` `visualization` が含まれる（FR-3, FR-7）
- TC-006: 可視化ごとに色空間、軸、読み方、特徴がテキストで返る（FR-4）
- TC-007: HTMLとJSONで主要色と要約文が整合する（FR-2, FR-3, FR-5）
- TC-008: APIドキュメントから入力形式と制限値を確認できる（FR-7）

## 異常系

- TC-101: 非対応画像形式の入力時にバリデーションエラーを返す（FR-1, FR-7）
- TC-102: 壊れた画像データの入力時にデコードエラーを返す（FR-1, FR-7）
- TC-103: `4.0 MB` 超過の入力時に `413` を返す（FR-8）
- TC-104: HTML利用時でも解析失敗が人間に判別可能な形で表示される（FR-2, FR-7）
- TC-105: `12 MP` 超過画像の入力時に制限エラーを返す（FR-8）
- TC-106: IPのレート制限超過時に `429` を返す（FR-8）

## 非機能

- TC-201: 同一画像を複数回解析して主要な集計値が一致する（NFR-2）
- TC-202: 解析完了後に元画像が永続保存されない（FR-6, NFR-1）
- TC-203: 構造化HTMLから見出しと主要項目を抽出できる（NFR-3）
- TC-204: 最新デプロイのJSONレスポンスをクライアントが解釈できる（NFR-4）
- TC-205: `15 秒` を超える解析がタイムアウトとして失敗する（NFR-6）
