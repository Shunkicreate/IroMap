# Vision

## 背景

IroMap は、写真の色構造を理解するための分析プロダクトです。
色空間可視化は目的ではなく、写真分析機能を成立させるための基盤と位置づけます。

## 目指す状態

- 写真の色分布・色距離・色面積比を可視化できる
- 写真ごとの色構造を比較し、特徴を説明できる
- 分析結果から改善アクションにつながる診断コメントを生成できる
- 機能ごとに要求と設計が追跡できる（SDD）

## プロダクト定義
>
> Understand the color structure of your photos

写真の色構造を可視化する分析ツールとして設計する。

- README に機能詳細を蓄積し続けること
- 実装依存の仕様管理
