# Glossary

- 色空間: 色を数値で表現するための座標系（例: RGB, HSL, CIELAB）
- スライス: 色空間の1軸または複数軸を固定して切り出した断面表示
- 変換: ある色表現から別の色表現へのマッピング
- Color Structure Analysis: 写真内の色分布・色面積比・多様性を定量化する分析
- 技術MVP: 実装基盤の成立を目的とした最小実装（IroMapでは RGB 可視化基盤）
- 機能MVP: ユーザー価値を提供する最小機能群（IroMapでは写真色構造分析）
- Colorfulness: 色の強さを定量化する指標（Hasler-Suesstrunk）
- Hue Entropy: 色相の多様性を示す指標
- SDD: Specification Driven Development。仕様を先に定義して実装を進める開発手法
- ADR: Architecture Decision Record。技術判断の記録
